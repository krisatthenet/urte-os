Under the hood

A real, POSIX-conformant OS core

URTE isn't a metaphor — it ships an actual operating-system kernel. Generated from a Capella MBSE model, POSIX-compliant by IEEE Std 1003.1-2024, with a Rust rewrite carrying its own pipeline language and SQL mesh.

// kernel-rs

Rust kernel

Memory-safe rewrite of the core: scheduler, semaphores, typed scale-aware IPC channels and a TRL-Pull scheduler.

// POSIX.1-2024

100% POSIX additive

Every URTE call follows POSIX conventions — real file descriptors usable with mmap, fstat, close. Unload the module → fully conformant.

// UPL

Pipeline language

A built-in compiler (lexer → parser → IR) and bytecode VM for the URTE Pipeline Language, executing against live kernel state.

// mesh.rs

In-kernel SQL mesh

Channels carry SQL request strings executed over the entity schema — RwLock reads run truly parallel, semaphore-bounded.

// guardrail.rs

Guardrails as syscalls

Size / TRL / scale guardrail masks enforced on every send; Ethics Board veto wired into the kernel decision path.

// MBSE

Model-driven

Interface & structure generated from the urtecore Capella model — Arcadia / SysML extension, single source of truth.
